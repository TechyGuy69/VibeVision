🌌 VibeVision & VibePlanner

An elite, AI-powered dual-engine travel curator
Seamlessly toggle between discovering instant local spots and generating bespoke, multi-day itineraries — all based purely on vibes.

🚀 The Vision

Traditional travel apps ask: “Where do you want to go?”
We ask: “How do you want to feel?”

Built for the modern explorer, this project introduces two powerful, interconnected engines:

🔍 VibeVision (Explore Mode)

Instantly scans any global neighborhood to discover places that match your exact mood.
Examples:

“Quietly Local Curated Cafes”
“Brightly Busy Nightlife”
🧠 VibePlanner (Pro Mode)

An AI-powered travel agent that crafts cinematic, day-by-day itineraries — complete with intelligent “Vibe Checks” for each moment.

✨ Premium Features
🌗 Dynamic Vibe Engine

The UI transforms based on your mood:

Quietly Local → Deep space gradients + glowing cyan particles
Brightly Busy → Crisp daylight glassmorphism + indigo fireflies

Powered by Google Gemini 2.5 Flash:

Converts raw location data into poetic, 2-sentence vibe summaries

💻 Running Locally
1️⃣ Clone the Repository
git clone https://github.com/your-username/vibevision.git
cd vibevision
2️⃣ Install Dependencies
npm install
3️⃣ Setup Environment Variables

Create a .env.local file in the root directory:

NEXT_PUBLIC_GEOAPIFY_KEY=your_geoapify_api_key_here
GEMINI_API_KEY=your_google_gemini_api_key_here
4️⃣ Start Development Server
npm run dev

Open 👉 http://localhost:3000
